Conocimiento en diversos lenguajes:

Java es un lenguaje de programación orientado a objetos (POO) que se utiliza principalmente para desarrollo backend. Este el primer lenguaje de programación que aprendí y mi favorito.

HTML es un lenguaje marcado que junto con CSS y JavaScript proporcionan un ecosistema clave para el desarrollo de aplicaciones web y páginas web. Gracias a HTML puedes ver este portfolio.

Python es un lenguaje muy versátil, normalmente se utiliza junto con herramientas de inteligencia artificial para 'Data Analysis' y scripts avanzados para la automatización de tareas.

Powershell también es POO como Java pero este trabaja directamente con el sistema operativo de Windows lo que te da una gran libertad para controlar tu ordenador y sus funciones
